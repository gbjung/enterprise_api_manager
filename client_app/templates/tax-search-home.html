Using the API

The API has been designed for use by developers. You will need basic knowledge of REST APIs, JSON, and HTTPS.

Getting Started

New Customers:
  • To receive API credentials and start shipping with us, please contact our local sales team here or talk to your local sales representative. Please provide the name of your company name and a contact person (first name, last name and e-mail address).
  • Once everything is settled, you will receive access to the sandbox for testing.
  • After successful testing you will receive credentials for the production environment.
  • Simply create your API request, using a valid access token required as described in our API reference documentation to submit shipments or get tracking information.
Existing Customers:
  • To receive API credentials please contact to your local sales representative. Please provide your Deutsche Post account number, your company name, and a contact person (first name, last name and e-mail address).
  • You will receive access to the sandbox for testing.
  • After successful testing you will receive credentials for the production environment.
  • Simply create your API request, using a valid access token required as described in our API reference documentation to submit shipments or get tracking information.
