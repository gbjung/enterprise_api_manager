Reference Documentation

This API provides an interface for our search service across the Bloomberg Tax product. It enables open integration channels for our customers and partners.

Search All

GET/tax/search/all/{query_term}

Search a query across the entire Tax product, including portfolios, codes and regulations, key rulings, related analysis, practice guides, and tables
